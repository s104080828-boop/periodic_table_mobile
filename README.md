# 元素週期表互動式學習（學習 / 闖關雙模式）

## 檔案說明
- `index.html`：首頁，分流到學習或闖關
- `study.html`：學習模式
- `challenge.html`：闖關模式
- `detail.html`：元素詳細頁
- `data.js`：元素資料
- `script.js`：學習模式互動
- `detail.js`：詳細頁互動
- `challenge.js`：闖關模式互動
- `style.css`：全站樣式

## GitHub Pages 部署
1. 建立一個新的 repository
2. 把這個資料夾內所有檔案上傳到 repository 根目錄
3. 到 `Settings > Pages`
4. 選 `Deploy from a branch`
5. Branch 選 `main`，資料夾選 `/root`
6. 儲存後等待 GitHub 產生網址

## 建議
- 手機與平板都可使用
- 學習模式適合先建立概念
- 闖關模式適合複習與自我測驗
